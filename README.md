# brightlearn-de-exam-Remofilwe-Mamabolo
Brightlearn capstone project

In this capstone project I was provided with a raw csv file as my data.  I need to clean and normalise the data so it is usable.
The data was placed in a staging database and was cleaned then put into the exam database.
SSIS etl packages were created to make these steps possible.
The database schema is snowflake because it provided the best normalisation.
